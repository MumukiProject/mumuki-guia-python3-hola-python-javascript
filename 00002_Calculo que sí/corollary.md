En programación a estos chirimbolos que utilizamos para hacer cálculos se los llaman _operadores_ y se escriben ligeramente diferentes:

|Operación	     | Operador  | Ejemplo  | 
|-------------	 |----------	|-------  |
|  Suma           | `+`       | `4 + 9`  | 
|  Resta          | `-`       | `12 - 2` | 
|  Multiplicación | `*`       | `2 * 3`  | 
|  División       | `/`       | `16 / 2` |
